# 1. Identity

## Project Name

BoardPrep

## Tagline

Prepare with Confidence.

## Mission

To provide Filipino learners with a modern, reliable, and continuously improving board examination preparation platform.

---

# 2. Brand Personality

BoardPrep should feel:

- Professional
- Trustworthy
- Encouraging
- Simple
- Modern
- Organized
- Student-focused

The platform should never overwhelm learners with unnecessary complexity.

Instead, it should guide users step by step toward examination readiness.

---

# 3. Design Philosophy

BoardPrep follows the principle:

> **Learn. Practice. Improve. Repeat.**

Every screen should help learners move naturally through this cycle.

---

# 4. Design Language

The interface should be:

- Clean
- Minimal
- Mobile-first
- Fast
- Accessible

Avoid unnecessary animations or visual clutter.

Learning should always remain the primary focus.

---

# 5. Tone of Voice

BoardPrep communicates in a tone that is:

- Friendly
- Professional
- Supportive
- Clear
- Encouraging

Explanations should teach rather than simply state whether an answer is correct or incorrect.
